CollabMaps
====================
A very simple webinos Google Maps collaborative editor.

### Description
A collaborative map viewing application with annotation features. Users can navigate to various places, add markers and their views get synchronized. Supports having different view modes, which means that one user can have satellite view while the  other user navigates using map view. It uses app2app and geolocation APIs
